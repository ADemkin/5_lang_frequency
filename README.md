# Frequency Analysis of Words

This script runs a frequency analysis for words in a given text and returns top 10 most frequent words.

Usage: 
```
python3 lang_frequancy.py [filename]
```

Example:
```
python3 lang_frequancy.py hamlet.txt
the
and
i
to
of
a
you
my
that
in

```
